# Henry's Optimized Dotfiles
A lightweight, high-performance Hyprland setup for Arch Linux.

## Features
- **Hyprland v0.45+** (Fixed shadow syntax)
- **Waybar + Cava** (Integrated audio visualizer)
- **Pywal Colors** (Global theming from wallpaper)
- **Minimalist** (No bloat, single-file configs where possible)

## Installation
```bash
git clone https://github.com/HenryScreee/dotfiles.git
cd dotfiles
./install.sh
```
